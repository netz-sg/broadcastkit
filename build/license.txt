WILLKOMMEN!

BroadcastKit ermoeglicht es dir, professionelle Overlays fuer 
OBS Studio zu erstellen und zu steuern - alles lokal auf 
deinem Computer.


FEATURES:
---------
[+] Lower Third Overlays
    3 Premium Styles: Clean Pro, Broadcast News, Esports HUD
    Mit Avatar-Support und animierten Effekten

[+] Now Playing Widget  
    Zeigt automatisch das aktuelle Spiel mit Cover an
    2 Styles: 3D Card, Cinematic Fullwidth
    Automatische Spielerkennung via RAWG API

[+] Social Media Widget (NEU!)
    Rotierende Social Media Cards fuer deine Kanaele
    Twitch, YouTube, TikTok, Instagram, Twitter, Discord
    Individuelle Anzeigedauer pro Plattform
    Auto-Rotation mit konfigurierbarem Intervall

[+] OBS Integration
    Auto-Connect beim Start
    Browser Source Refresh aus der App
    Echtzeit-Steuerung aller Overlays


BETA-HINWEIS:
-------------
Diese Software befindet sich noch in der Entwicklung!

* Einige Funktionen koennen noch fehlerhaft sein
* Feedback und Fehlermeldungen sind willkommen
* Updates koennen Breaking Changes enthalten


SYSTEMANFORDERUNGEN:
--------------------
* Windows 10/11 (64-bit)
* OBS Studio mit WebSocket Plugin (v5.x)
* Internetverbindung fuer Spielerkennung (RAWG API)


Mit der Installation akzeptierst du, dass es sich um 
Beta-Software handelt und keine Garantie fuer fehlerfreien 
Betrieb gegeben wird.
